# Chicago Indian Arts - Project Context

This project is a premium website for **Chicago Indian Collective Arts**.

## Project Status: PHASE 2 COMPLETE
The website is fully rebranded, mobile-responsive, and technically stable.

### **What is Done (Finished)**
- **Rebranding:** All URLs, logic IDs, and text origins moved from old names to new festival titles.
- **UI/UX:** Navbar optimized with accordion logic, bolder text, and deep shadows.
- **Responsiveness:** All pages (including Team and Programs) tuned for perfect scaling on mobile and tablets.
- **Integrations:** The **CODE** for Stripe, Telegram, and Google Sheets is 100% written and ready.

## Technical Stack

- **CMS:** Payload CMS 3.x (MongoDB)
- **Styling:** Tailwind CSS v4
- **Images:** Cloudinary CDN
